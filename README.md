# Watch Later CLI

A simple command-line tool to manage a personal "Watch Later" video queue. Add videos with their duration, track how much you've watched, and update progress as you go — all from the terminal.

## Features

- Add a video with its name and duration
- View your full watch-later list
- Update the remaining duration when you've partially watched a video
- Remove videos once fully watched

## Why I Built This

I personally end up saving a lot of videos to "Watch Later" and never tracking how much I've actually watched. This CLI app solves that by letting me log videos manually and update their remaining duration, so I always know exactly where I left off.

## Tech Stack

- **Language:** Python
- **Interface:** Command-Line (CLI)

## How to Run
